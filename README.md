# SampleApp - небольшой клон твиттера

Небольшое API для управления пользователями и ролями.

## Функционал
- Создание, редактирование, получение и удаление пользователей
- Создание ролей и привязка их к пользователям
- Возврат корректных HTTP-кодов (200, 201, 404)
- Репозиторный паттерн и базовая валидация данных

## Технологии
- ASP.NET Core Web API
- C#
- In-memory Singleton Repository

## Запуск
1. Клонировать репозиторий:
```bash
git clone https://github.com/username/project-name.git
```
2. В командной строке выполнит:
```bash
dotnet run
```
